Axell AI Contest 2025用データセット

* データセットの内容について
本データセットはアクセル社が飲料缶を撮影して作成したオリジナルのデータセットです。
91種類の飲料缶を1~5本並べて撮影した画像約11000枚で構成されます。

* データセットの形式について
データセットは coco フォーマットのアノテーションファイルと画像ファイルで構成されます。

dataset
+ README.txt    # 本資料
+ LICENSE.pdf   # ライセンス情報
+ annotations
| + train.json  # 学習用データ
+ images
  + T1.jpg      # 画像データ
  + T2.jpg
  + ...
  + TXXXX.jpg

アノテーションファイルの構造は次のような構造となります。

{
  "images": [
    {
      "id": <画像のID>,
      "file_name": <画像のファイル名 例:"T1.jpg">,
      "height": <画像の高さ ピクセル単位>,
      "width": <画像の幅 ピクセル単位>
    }
   ],
  "annotations": [
    {
      "id": <アノテーションのID>,
      "image_id": <画像のID>,
      "category_id": <カテゴリー(缶の種別)のID>,
      "bbox": [
        <缶の矩形 左上のx座標 ピクセル単位>,
        <缶の矩形 左上のy座標 ピクセル単位>,
        <缶の矩形の幅 ピクセル単位>,
        <缶の矩形の高さ ピクセル単位>
      ]
    }
  ],
  "categories": [
    {
      "id": <カテゴリー(缶の種別)のID>,
      "supercategory": <スーパーカテゴリー "can" もしくは "bottle">,
      "name": <カテゴリー名 例： "light_sugar_coffe1_185ml">
    }
  ]
}

缶のカテゴリーは銘柄別の名称("light_sugar_coffe1"など)と容量("185ml"など)で構成されます。
同一商品でも容量が異なる場合は別のカテゴリーとして扱う必要があります。
